# Docker Images

Custom and rebuilt Docker images for personal use.

## About

This repository contains Dockerfiles for building custom Docker images, primarily for personal projects and self-hosted services.

## License

Apache 2.0

## Usage

Each subdirectory contains a Dockerfile and any necessary configuration files for building a specific image.

To build an image:

```bash
cd <image-directory>
docker build -t <image-name>:<tag> .
```
